# Juego de velocidad 

Un juego interactivo de velocidad de escritura desarrollado con HTML, CSS y JavaScript Vanilla.

## ¿Cómo funciona?
El juego genera palabras aleatorias en pantalla que el usuario debe escribir correctamente en el menor tiempo posible. Cada palabra correcta suma puntos al contador, y al agotarse el tiempo de la cuenta regresiva, el juego finaliza mostrando el puntaje total obtenido con la opción de reiniciar la partida.

## Tecnologías utilizadas
* **HTML**: Estructura general de la interfaz de juego y contenedores de datos.
* **CSS**: Diseño responsivo y estilizado centrado con sombras y tipografía limpia.
* **JavaScript**: Manejo de temporizadores (`setInterval`), lógica de arrays aleatorios y captura de inputs en tiempo real.
